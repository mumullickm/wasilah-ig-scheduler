# Wasilah Quran-60 cloud publisher

GitHub Actions cron that publishes the 60 Quran Ayah reels to Instagram (and mops up the last Facebook posts) on schedule, independent of any local machine. Runs 03:00 + 13:00 UTC (09:00 + 19:00 Dhaka). Token stored as the META_PAGE_TOKEN repo secret. State in posted.json.
